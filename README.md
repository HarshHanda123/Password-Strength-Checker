# Password Strength Checker

This project is a Python-based application that checks the strength of a password.

# Features
- Checks password length
- Detects uppercase and lowercase letters
- Detects numbers and special characters
- Provides strength rating (Weak, Medium, Strong)
- Gives suggestions to improve password

# Technologies Used
- Python

# How to Run
1. Run the Python file
2. Enter a password
3. View the strength and suggestions
4. You can also view your score based on your passsword

# Example
Input: Abc@1234  
Output: Strong Password
